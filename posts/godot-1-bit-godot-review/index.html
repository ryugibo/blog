1 Bit Godot 코스 리뷰

March 11, 2021 · 2 분

https://www.heartgamedev.com/1-bit-godot-course-youtube

작년에 할인할 때 샀던 코스를 지금 마무리했다.

내용이 많거나 어려운건 아닌데 미루다가 godot 3.2.4 가 나오기 전에는 해두는게 좋을 것같아서 얼른 마쳤다. 강의를 따라하다보면 3.2.2에서 3.2.3로 마이너 버전 업데이트가 맞나 싶을 정도로 차이점이 눈에 보여서 3.2.4 에서 그대로 따라할 수 있을지 모르겠다.

강의 내용은 간단한 게임 4개를 만들어보는데 마지막 Metroidvania 프로젝트가 가장 분량이 많고 앞선 강의에서 다뤘던 내용이 포함되어있기도 해서 어느 정도 다른 엔진 경험이 있거나 하면 Metroidvania만 따라해보는것도 충분할 것 같다.

특별히 따라하면서 놀랐던건 보스 애니메이션 잡는 부분. godot에서 분리되어있는 2D 리소스에 본을 엄청 쉽게 붙이는 기능. 다른 엔진에도 이정도는 쉽게 지원할지 모르겠지만 보스 애니메이션 잡는 과정에서 대충 스프라이트만 골라잡고 버튼 딱딱 누르면 본이 촵 연결되는것 보고 놀랐다. Spine 이나 Spriter 같은 외부 프로그램 없이 godot 에서 2D 애니메이션은 다 잡을 수 있지 않을까..?

그 외에 이동, UI 등의 로직은 heartbeast 선생님 유튜브에도 있는 많은 무료 강의에서 볼 수 있는 내용과 겹치는 내용도 있어보인다.

유료 강의로서 장점은 heartbeast 디스코드에 참여가 가능해서 막히면 물어보거나 기존에 물어봤던 질문들을 검색해볼 수 있다는 점.

물론 영어로..

자기가 만들고 싶은 목표가 확실하고 강의에서 만들어보는 Metroidvania 와는 방식이 다를 경우, godot관련 기본적인 내용을 배우고 싶은 경우는 유튜브만 보면서도 충분히 배울 수 있을 것같다.

강의와는 별개로 godot 3.2.3 문제라고 생각되는데, 한글 에디터로 강의를 진행하다보면 어느 순간부터 Player.tscn 만 열면 not all arguments converted during string formatting 에러가 두세개씩 찍힌다.

이 에러는 heartbeast 선생님이 github에 올려놓은 코드를 열어도 발생하는 문제인데 heartbeast 디스코드에는 비슷한 에러 질문 기록도 없고, 구글을 검색해보면 파이썬 관련 에러나 godot 관련해서는 문자열에 &를 넣은 경우정도가 나온다.

결론은 에러가 신경쓰이지 않으면 그냥 해도 문제는 없으나 다른 에러가 발생했을 때, 에러 원인 파악에 방해가 될 수 있으므로 그냥 영어 에디터를 쓰는게 맘이 편할 것같다.
